# Contexto: Sistema para Casamentos

Propósito: Gerenciar casamentos.

Fluxos principais:
- Convidados: Adição, RSVP, restrições alimentares, alocação de assentos.
- Relacionamentos: Vínculos entre convidados.
- Espaço: Mesas (capacidade, formato) e Assentos.
- Financeiro: Orçamento, despesas, vencimentos, valores pagos.
- Autenticação: Registro/login de casais.

Atores:
- Usuário (role: couple, plan: free/premium): Dono do casamento. Gerencia dados.
- Convidado: Entidade passiva. Status RSVP gerido pelo sistema.
